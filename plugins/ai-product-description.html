AiProductDescription

Ein-Klick-Produktbeschreibungen direkt im Shopware-Admin. Claude oder ChatGPT wählbar, zwei Sprachen, drei Schreibstile – die API-Keys verlassen nie den Server.

GitHub ansehen →
// screenshots
// features

Was es kann.

  • Ein-Klick-Button direkt über dem Beschreibungsfeld im Produkt-Editor
  • Zwei KI-Anbieter wählbar – Claude (Anthropic) und ChatGPT (OpenAI)
  • Zweisprachig – Deutsch und Englisch
  • Drei Schreibstile: Professionell, Casual, SEO-optimiert
  • API-Calls laufen serverseitig – die Keys verlassen nie den Browser
  • Automatik-Modus liest Produktname, Meta-Infos und Kategorien selbstständig aus
// konfiguration

Konfiguration.

Einstellung Standard Beschreibung
Anthropic API KeyemptyAktiviert die Verbindung zu Anthropic Claude
OpenAI API KeyemptyAktiviert die Verbindung zu OpenAI ChatGPT
Mindestens ein Key—Mindestens einer der beiden Keys ist erforderlich
// installation

Installation.

Plugin nach custom/plugins/ kopieren, installieren und die Admin-Oberfläche neu bauen:

michael@shop:~
$ cp -r AiProductDescription custom/plugins/MichaAiProductDescription
$ bin/console plugin:refresh
$ bin/console plugin:install --activate MichaAiProductDescription
$ bin/build-administration.sh
$ bin/console cache:clear
// verwendung

So wird es benutzt:

  1. Produkt im Admin öffnen und zum Beschreibungsfeld scrollen
  2. Auf „AI Beschreibung generieren" klicken
  3. Anbieter (Claude/ChatGPT), Sprache und Schreibstil wählen
  4. Automatisch oder manuell mit eigenen Stichpunkten generieren lassen
  5. Ergebnis prüfen und optional direkt im Editor anpassen
  6. Übernehmen – fertig, ohne die Seite zu verlassen
